# schronisko-baza
Projekt bazy danych schroniska - moduł pierwszy z dwóch

Relacyjna baza danych zaprojektowana na potrzeby kompleksowej obsługi schroniska dla zwierząt lub placówki weterynaryjnej. 
Architektura systemu pozwala na ewidencjonowanie zwierząt, śledzenie ich historii medycznej, zarządzanie procesem przyjęcia oraz monitorowanie zachowań.
